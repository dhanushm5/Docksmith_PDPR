# Docksmith_PDPR

Project layout:

- `docksmith/`: Linux-focused no-Docker variant
- `docksmith-linux/`: duplicate Linux workspace kept for isolation/testing
- `docksmith-macos/`: macOS/Docker-based wrapper variant

Use `docksmith/` on Linux and `docksmith-macos/` on macOS if you want the platform-specific workflows side by side.
